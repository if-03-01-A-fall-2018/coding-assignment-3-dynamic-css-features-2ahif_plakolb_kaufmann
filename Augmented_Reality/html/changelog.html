How to use NGINX (Assumes that you have a fully setup https nginx server!):

Go into terminal type 'sudo service nginx start'. Go into your browser type https://localhost on the pc, type 'ip addr show' to see the IP you have to use on your mobile, should be under eth1 inet 255.255.255.255 do not use any port(no :8080 behind localhost).
